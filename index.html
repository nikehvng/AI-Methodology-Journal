Warum KI jetzt?

Warum Künstliche Intelligenz jetzt?

Einführung

Künstliche Intelligenz gibt es schon seit Jahrzehnten. Doch erst jetzt erlebt sie ihren Durchbruch. Warum? Weil mehrere Entwicklungen gleichzeitig zusammenkommen.

Inhalt

  • Daten: Enorme Mengen an Daten durch Internet und Digitalisierung
  • Rechenleistung: Leistungsstarke GPUs und Cloud Computing
  • Algorithmen: Fortschritte im Machine Learning und Deep Learning
  • Anwendungen: KI wird wirtschaftlich nutzbar (z. B. Chatbots, Bilderkennung)

Merksatz

KI wird jetzt groß, weil Daten, Rechenleistung und Algorithmen gleichzeitig ein kritisches Niveau erreicht haben.

Machine Learning

Machine Learning

Einführung

Machine Learning ist ein Teilgebiet der Künstlichen Intelligenz. Statt feste Regeln zu programmieren, lernt ein System selbstständig aus Daten.

Inhalt

  • Supervised Learning: Lernen mit gelabelten Daten (z. B. Spam-Erkennung)
  • Unsupervised Learning: Muster erkennen ohne Labels (z. B. Clustering)
  • Reinforcement Learning: Lernen durch Belohnung und Bestrafung
  • Ziel: Vorhersagen treffen oder Entscheidungen optimieren

Merksatz

Machine Learning bedeutet: Systeme lernen aus Daten, statt explizit programmiert zu werden.
